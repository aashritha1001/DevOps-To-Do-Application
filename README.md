# DevOps-To-Do-Application

## Overview
This project is a simple To-Do web application developed using DevOps practices. It allows users to manage daily tasks efficiently.

## Features
- Add new tasks
- Update tasks
- Delete tasks
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Git
- Docker

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Run the application in your browser or using a local server.
